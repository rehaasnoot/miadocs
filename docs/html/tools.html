tools.animate module¶

tools.animate.main()¶
tools.animate.performance(playerName=None, instrumentName=None, instrumentTrackIds=None, voxTrackId=None)¶

Top-level function designed to be run from within a blender text window.

Will detect a player, instrument and data-file, then execute MIA to produce the animation.

playerNamestr, optional

The name of the player model (default is None)

instrumentNamestr, optional

The name of the instrument model (default is None)

instrumentTrackIdsint | str | ‘[int,int,..]’ , optional
Override the TrackId custom property of the instrument

to indicate which tracks are to be played (default is None)

voxTrackIdint , optional
Override the TrackId custom property of the player

to indicate which tracks are to be sung. (default is None)
