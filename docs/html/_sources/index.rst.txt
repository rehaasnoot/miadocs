Musical Instrument Animator
===========================

**MIA** is software that automates the heavy-lifting necessary to animate performing musical instruments in `Blender`_. `MIA`_ is designed to animate any character playing any song on any instrument.
And by encoding voicing in a `MIDI`_ file, `MIA`_ also supports singing.
